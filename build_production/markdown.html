Using Markdown to add code syntax highlighting in HTML emails

This email template illustrates how code syntax highlighting in HTML email looks like. It was created with PrismJS, Synthwave '84 theme, and Markdown fenced code blocks in Maizzle.

For example, here's a block of JavaScript code:

function foo(bar) {
  var a = 42,
    b = 'Prism';
  return a + bar(b);
}

And here is some HTML markup:

<table class="w-full">
  <tr>
    <td class="sm-w-1-2 sm-inline-block bg-white">
      <p class="m-0 text-base text-gray-800">A simple paragraph</p>
    </td>
  </tr>
</table>
